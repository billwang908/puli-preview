## 登入

## 功能

- 必須登入後才可使用系統。
- 已載入 `C:\Users\user\Desktop\Cus.xls`，並拆分成三個 table：
  - 產品資料：取自「商品」工作表。
  - 客戶資料：取自「客戶」工作表。
  - 銷售資料：取自「銷貨縮表」工作表。
- 三個 table 皆支援 CRUD：新增、修改、刪除、查詢。
- 查詢條件依 table 調整；銷售資料支援客戶姓名、產品編號/型號、日期區間。
- 文字條件採模糊比對。
- 報表：月銷售趨勢、淡旺季分析、熱門產品排行。
- 支援手機版操作，表格在手機上會轉成分段資料卡。
- 可重新上傳 Excel/CSV，重新拆分三個 table。

## 注意

目前版本為純前端展示與單機使用版，資料儲存在瀏覽器 localStorage。
若要正式多人上線，建議改接後端登入驗證與資料庫。
